Installing Bash for Windows:- https://msdn.microsoft.com/en-gb/commandline/wsl/about

Re-installing Bash for Windows:-
Warning, this will delete your entire Bash on Windows environment.
* Backup any settings or data WSL contains first.
* Open a Command Prompt Window
* lxrun /uninstall
* lxrun /install

Installing snapcraft in Bash on Windows:-
* Search the start menu for “Bash”
* Click the Ubuntu icon
* In the Bash on Windows environment, run “sudo apt install snapcraft”
