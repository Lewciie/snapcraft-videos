Rozpoczynanie pracy ze Snapcraft

Instalacja snapcraft'a jest szybka i łatwa . To wideo zademonstruje jak to zrobić.

Za pomocą paczki snap: sudo snap install snapcraft --beta --classic
Za pomocą dockera: sudo docker run -v $PWD:$PWD -w $PWD snapcore/snapcraft snapcraft
Zaloguj się do sklepu: snapcraft login

Zarejestruj się w sklepie: http://dashboard.snapcraft.io/
