Get started with Snapcraft

Installing snapcraft easy quick and easy. This video shows how.

Install the snap: sudo snap install snapcraft --beta --classic
Install via docker: sudo docker run -v $PWD:$PWD -w $PWD snapcore/snapcraft snapcraft
Login to the store: snapcraft login Store sign up: http://dashboard.snapcraft.io/
